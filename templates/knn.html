K-Nearest Neighbor

In short, the K-nearest neighbor model is a classification technique, also used for regression at times. This method looks at the data presented and puts data points in groups that it best thinks correlates to each group. The K determines the amount of neighbors it is closest to, and uses it to classify the data point, so it's a rather straightforward method compared to others, but it takes a longer time the higher the K value becomes.
