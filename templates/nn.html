Neural Network

A neural network is a simplified model of the way the human brain processes information. It works by simulating a large number of interconnected processing units that resemble abstract versions of neurons. [1] The basic units are neurons, which are typically organized into layers. There are typically three parts in a neural network: an input layer, with units representing the input fields; one or more hidden layers; and an output layer, with a unit or units representing the target field(s). In this section, we explored the outcomes of neural network models with 1 and 2 hidden layers. We refer to the models with 2 hidden layers as deep learning models.

Model Parameters

The cleaned Amex Dataset has 32 features. We first constructed the models using all features. The neural network with 1 hidden layer had 3,502 trainable parameters, and the deep learning model with 2 hidden layers had 13,602 trainable parameters. Since other models in this project have already identified the most important 5 features in this dataset, we used the top 5 features to construct simplified neural network models. The neural network with 1 hidden layer had 802 trainable parameters, and the deep learning model with 2 hidden layers had 10,902 trainable parameters. Model Outcomes

For models that utilized all 32 features, accuracies increased progressively with the increased epochs. For the model with 2 hidden layers, the loss for the testing data increased dramatically compared to the model with 1 hidden layer, suggesting possible overfitting with the deep learning model.

For models that utilized the top 5 features, the performance of the models with 1 and 2 hidden layers are very similar. Accuracy and loss for the training and testing data stabilized quickly for both models.
