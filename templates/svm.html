Support Vector Machine

Support vector machine, also known as SVM, is a supervised learning model that is widely used for classification problems. SVM models utilize a hyperplane, a decision boundary that helps classify data points based on their number of features. By finding a hyperplane that maximizes distance between data points, future data points can be classified with more accuracy. Thus, SVM models are very advantageous when working with high dimensional data.

Implementation of the AMEX Dataset

The training AMEX dataset contains 33 features that will be used by the SVM model to accurately predict default or no-default. After fitting the training data, the SVM model scored an accuracy of 98%. Upon doing a classification report, it was found that the model has a precision of 98% when classifying a datapoint as no-default and a precision of 100% when classifying a datapoint as default.
